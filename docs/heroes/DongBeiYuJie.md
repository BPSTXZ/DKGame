# 东北雨姐 (DongBeiYuJie)

## 基础信息
- **名称**: 东北雨姐
- **最大HP**: 100
- **基础移速**: 60
- **主题色**: `#e91e63`
- **碰撞伤害**: 2点 (冷却1秒)

## 外观与朝向规则
1. **球体贴图**: 使用 `assets/img/B16.png` 作为球体填充贴图，显示在圆形裁剪区域内。
2. **朝向规则**: 身体始终朝向敌方单位。
3. **额外贴图层**: 使用 `assets/img/B17.png` 叠加在球体上用于表现技能大汗脚。默认状态下重叠覆盖，触发技能时沿朝向拉伸延展。

## 技能一：带派大汗脚
- **定位**: 持续压制技能。距离越远脚越大，伤害与范围越强。方向与尺寸锁定，可被走位规避。
- **触发与冷却**: 
  - 触发频率：基础触发间隔 2秒。
  - 持续时间：3秒。
  - 冷却规则：技能结束后进入 2秒 冷却。
  - 触发条件：未在持续中、未冷却中，且未被控制。
- **方向与尺寸动态追踪规则**:
  - 锁定方向：持续指向敌方单位的方向。
  - 锁定长度：持续追踪并等于自身到敌方单位的距离（L）。
  - 大汗脚在伸出后，会像橡胶一样实时随敌方的移动而改变方向和长度，不再是可以被轻易走位躲避的固定陷阱。
- **范围判定**:
  - 采用线段到点的距离检测，命中判定为距离 <= R + 敌方半径。
  - 随长度增长，半径 R = 10 ~ 30。
- **伤害分档 (以屏幕 600 为 S)**:
  - **小汗脚** (L < 0.28S): 每秒 2点 (每 tick 0.4)。
  - **中汗脚** (0.28S <= L < 0.48S): 每秒 6点 (每 tick 1.2)。
  - **大汗脚** (L >= 0.48S): 每秒 9点 (每 tick 1.8)。
- **动画表现**:
  - 前摇 (0.15s): 轻微挤出。
  - 伸展 (0.25s): 线性伸长。
  - 持续 (3s): 呼吸抖动。
  - 收回 (0.2s): 快速收回。
- **视觉特性**:
  - 利用非均匀缩放分为三段绘制，模拟近大远小的透视效果。
  - 命中敌方时不断生成蓝色的冷汗粒子特效。
  - **气味特效**: 汗脚在伸出和持续期间，足部会不断向斜上方散发出绿黄色、半透明的“气味云雾”粒子，强化视觉冲击力。

## 觉醒：四方带派
- **定位**: 强控场压迫，持续 5秒。
- **触发效果**: 从场地上下左右四周分别伸出一只中汗脚，持续 5秒。每只脚伤害为 **每秒 2点** (每 tick 0.4)。
- **生成位置与方向建议**: 四只脚从上、下、左、右四个边框的中点伸出。
- **动态追踪**: 觉醒的四只大脚不再指向固定中心点，而是**实时追踪敌方**，长度和方向会时刻调整以精准踩中敌方单位，让敌方无处可逃。
- **动画与逻辑**:
  - 觉醒期间暂停技能一的冷却。
  - 0.25秒红色预警。
  - 0.2秒迅速伸出。
  - 5秒持续压迫。
  - 结束时淡出并恢复正常状态。
