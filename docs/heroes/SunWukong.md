# 猴哥 (SunWukong)

**"俺老孙来也！"**

## 1. 英雄设定

*   **职业**：齐天大圣、金刚不坏
*   **战斗风格**：近战压制与大范围扫荡
*   **特性**：通过如意金箍棒进行范围输出，并依靠“金刚不坏”被动在关键时刻免疫多段伤害。觉醒后拥有全场压制力。

## 2. 基础属性

*   **最大生命值 (HP)**：100
*   **基础移速**：60
*   **主色调**：`#fbd73a` (明黄色)
*   **武器**：如意金箍棒

## 3. 技能机制

### 3.1 核心武器：如意金箍棒
金箍棒以扁平风（带有两端金箍高光、中间红色的圆角矩形）渲染，水平放置并围绕猴哥周身轨道进行**逆时针**圆周旋转。

*   **常态**：
    *   **长度**：单位半径的 3 倍。
    *   **转速**：360度/秒。
    *   **伤害**：命中敌方造成 5 点伤害（受攻击倍率影响）。
    *   **冷却**：对同一目标的命中结算有 0.2 秒内置冷却，防止重复刷伤。
*   **强化态 (如意效果)**：
    *   **触发**：每 4 秒自动触发一次。
    *   **连段限制**：当猴哥处于连段受击状态（包括被吸血鬼持续吸附/流血）时，强化态进度会暂停增长；连段伤害结束 0.5 秒后，若冷却完毕则会立即触发并重新计时。
    *   **表现**：转速提升至 1.5 倍，棍身会拉出 3 层向近端递减的半透明残影拖尾（呈现由多到少的视觉效果），爆发瞬间会向外侧散发金色粒子。
    *   **变长**：旋转的前半圈（0~180度）棍子远端向外线性伸长至最大 350 像素（覆盖大半个屏幕）；接着在最大长度旋转一整圈（360度）；最后半圈（180度）线性回缩至常态。近端始终锚定在角色周身轨道上。
    *   **抖动优化**：为了避免画面静止时的抽搐，棍身的弯曲抖动效果采用了基于角度的伪随机振荡（正弦函数叠加），而非纯随机数。
    *   **伤害**：强化态命中造成 10 点伤害，并触发金色冲击环与火花特效。

### 3.2 被动：金刚不坏之身
猴哥在受击时有概率触发护体金光，免疫伤害并规避敌方的连段攻击。

*   **连段判定机制**：系统设定了 0.5 秒的连段判定窗口（`comboHitWindow`）。短时间内的连续受击（或吸血鬼的持续吸血）均视为同一套连段。
*   **触发概率**：在承受一套连段的**第一下**伤害时，有 20% 概率触发。
    *   如果**触发成功**：爆出金色光晕，本次伤害免疫，并在后续整个连段窗口期内（只要连段不断）保持金刚不坏状态，完美吃下高频多段伤害。
    *   如果**触发失败**：本次连段的后续所有伤害都**不会**再进行概率判定，确保平衡性，避免高频多段攻击必定触发被动。

### 3.3 觉醒：大闹天宫
拾取觉醒石后，猴哥爆发出强大的全场扫荡能力。

*   **阶段 1 (弹射散开 - 1秒)**：
    *   猴哥爆出集中的金色云雾（30个粒子），向四周**均匀（呈120度夹角）**弹射出 3 个半透明分身。
    *   分身弹射带有**线性减速效果**（初始速度 1200，每秒衰减至最低 350），在场地上保持持续弹射运动。
*   **阶段 2 (等待蓄力 - 1秒)**：
    *   分身继续在场上不断碰壁反弹，棍子保持逆时针旋转蓄力。
*   **阶段 3 (四棍齐发 - 1圈)**：
    *   本体与 3 个分身**同时**触发强化态的如意效果。
    *   四根金箍棒在全场同步伸长扫荡，造成巨量范围伤害。（分身拥有独立的伤害冷却计时器，互不干扰）。
*   **阶段 4 (渐隐消失 - 1秒)**：
    *   攻击结束，分身在移动中逐渐降低透明度，并散成金色小光点消失。本体恢复常态。

## 4. 开发实现细节
*   代码文件：`src/game/entities/heroes/SunWukong.js`
*   物理碰撞：由于金箍棒是一头锚定一头甩出的倾斜状态（切线角 + 30度），碰撞检测使用了 `Physics.js` 中的 `checkLineCircleCollision`。
*   特效管理：大量使用了 `this.game.addParticle` 制造爆气、残影、火花和冲击环，并在觉醒阶段巧妙利用了 `Math.random()` 和坐标更新使分身表现更生动。
*   音效管理：
    *   **选择英雄**: `assets/audio/SunWukong/老孙来也.mp3`
    *   **常态击中**: `assets/audio/common/碰撞.mp3`
